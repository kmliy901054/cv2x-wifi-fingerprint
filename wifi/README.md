## 每行格式(一行一筆 scan)

```json
{
  "stamp": {"sec": 1778984035, "nanosec": 953887027},
  "wall_time": 1778984037.56,
  "pose": {"frame_id": "map", "x": 3.136, "y": 0.163, "z": 0.0,
           "yaw": 0.217, "quat": {"x":0,"y":0,"z":0.108,"w":0.994}},
  "scan_duration_ms": 3800,
  "aps": [
    {"ssid": "ASUS_A8_2G", "bssid": "F0:2F:74:E2:C4:A8",
     "rssi": -51, "channel": 1, "encryption": "WPA2"},
    ...
  ]
}
```

- `pose` 是該筆 scan 對應的機器人位置(map frame)。stamp 取 scan 中段時間,
  避免 ESP32 一輪掃描 ~3.8 s 期間移動造成的對齊偏移。
- `aps` 是該次掃到的 AP 清單;Lab 3 只保留出現 ≥10 次的 80 個 BSSID 當特徵。
